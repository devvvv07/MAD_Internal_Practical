# Smart Attendance Management App

## Overview
This is a Flutter-based Smart Attendance Management App designed to automate
and secure attendance tracking using QR codes and GPS-based validation.

## Features
- Role-based login (Admin, Instructor, Student)
- Session-based QR code attendance
- Time-bound QR validation
- GPS-based location verification
- Offline attendance capture with sync
- Admin dashboard with analytics and reports

## Technology Stack
- Flutter
- Firebase Authentication
- Firebase Firestore
- QR Code Scanner
- GPS Location Services

Under active development.
